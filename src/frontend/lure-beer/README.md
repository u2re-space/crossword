# LURE + Beer CSS edition

**Planning, Scheduling and Organizer Web Application with AI Tools**

This is the alternative intermediate version of the frontend implementation using LURE and BeerCSS library.

It is used to test the integration of the LURE and BeerCSS libraries.

## Used Frameworks and Libraries

- [LURE](https://github.com/fest-live/lure) - Reactive DOM framework
- [Core.TS](https://github.com/fest-live/core.ts) - Core utilities library
- [Object.TS](https://github.com/fest-live/object.ts) - Core reactivity library
- [DOM.ts](https://github.com/fest-live/dom.ts) - DOM utilities
- [Icon.TS](https://github.com/fest-live/icon.ts) - Icon library
- [Beer CSS](https://beer-css.github.io/) - Material Design CSS framework
- [Material Color Utilities](https://github.com/material/material-color-utilities) - Material Design color system
- [Phosphor Icons](https://phosphoricons.com/) - Icon library
- [SortableJS](https://github.com/SortableJS/Sortable) - Drag and drop functionality
